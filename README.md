# liemu

A RISC-V emulator written in C++ which can run simple RV32I programs for education.

## build

```bash
make
```

## run

```bash
make run
```

## liemu commands

- `help`: show cmd help
- `q`: quit
- `ls`: list source code  
- `clear`: clear screen
- `si`: single step
- `c`: continue
- `info`: show register or watch ponit info
- `x`: show memory content
- `w`: add watch point
- `d`: delete watch point
